# Codes-for-panel-ARMA-GARCH-model


The simulation and empirical codes for the paper ``Inference for the panel ARMA--GARCH model when both N and T are large'' are provided in the file. 


The stock volume data comes from Yahoo, downloaded by the R package tidyquant. 
The portfolio returns data comes from the website  https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/data_library.
